# Changelog

All notable changes to the Onplana Cursor plugin are documented here. This project follows [Semantic Versioning](https://semver.org/).

## [0.1.0] , 2026-06-25

Initial release.

- Bundles the remote Onplana MCP server (`https://mcp.onplana.com/mcp`, OAuth 2.1 + DCR + PKCE) with an Onplana project-management skill.
- `skills/onplana/SKILL.md`: product model, tool families, and common workflows (stand up a project, decompose into tasks/sprints/epics/milestones, check status & EVM, run risk analysis, log time, execute assigned work) with example prompts.
- Tool count stated as **247**, reconciled against the live server `tools/list`.
- Chevron logo (#2563EB) committed and referenced by relative path.
